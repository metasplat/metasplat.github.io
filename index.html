What is this?

Metasplat is an alternative to the Metasploit Framework, exclusively written in c and assembly, partly as a challenge and partly as means to improve low-level control over every operation you take. I started its development initially in 2018, and released it as an Open-Source project under the GNU License in 2022.

How to use?

Right now the only way to use Metasplat is to build from source since automation is still yet to be my prioritie. Only linux-based OS's are supported, and im not planning on adding Windows support any time soon.
